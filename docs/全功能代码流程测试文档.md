## 一、认证模块 (campus-auth) — 6 个端点

### 1.1 密码登录

```
POST /api/v1/auth/login
```

**调用链**：
```
前端 login/index.vue
  → axios.post('/api/v1/auth/login', { phone, password })
    → Gateway AuthGlobalFilter (白名单 /api/v1/auth/login → 放行)
      → AuthController.loginByPassword(@RequestBody Map<String,String> body)
        → AuthService.loginByPassword(phone, password)
          → SysUserMapper.selectOne(phone)           // MySQL campus_user.sys_user
          → BCryptPasswordEncoder.matches(pw, hash)   // 密码比对
          → buildTokens(user)                          // 签发 JWT
            → JwtTokenProvider.createAccessToken(userId, tenantId, userType, roles)
              → JWT Payload: { sub=userId, tid=tenantId, typ=userType, rol=roles, jti=uuid, exp=now+2h }
            → JwtTokenProvider.createRefreshToken(userId, tenantId)
              → JWT Payload: { sub=userId, tid=tenantId, exp=now+7d }
          → SysUserMapper.updateById(user)            // 更新 last_login_time
        ← return Map{ accessToken, refreshToken, userId, nickname, userType }
      ← Result.success(tokens)
    ← HTTP 200 { code:200, data:{ accessToken:"eyJ...", refreshToken:"eyJ...", userId:"169...", nickname:"张三", userType:"ADMIN" } }
  → localStorage.setItem('accessToken', ...)
  → userType='ADMIN' → router.push('/dashboard')
  → userType='STUDENT' → window.location.href='http://localhost:3001/student?token=...'
```

**📥 输入**:
```json
{ "phone": "13800138000", "password": "MyPass123!" }
```

**📤 输出** (成功):
```json
{
  "code": 200, "message": "success",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "userId": "1692345678901234567",
    "nickname": "张三",
    "userType": "ADMIN"
  }
}
```

**📤 输出** (用户不存在):
```json
{ "code": 1001, "message": "用户不存在", "data": null }
```

**📤 输出** (密码错误):
```json
{ "code": 1002, "message": "密码错误", "data": null }
```

✅ **预期**: BCrypt 密码匹配成功 → 返回双 Token → 前端存 localStorage → 按 userType 跳转

### 1.5 刷新 Token

```
POST /api/v1/auth/refresh
```

**调用链**：
```
前端 shared/utils/request.ts 拦截器
  → 401触发 → axios.post('/api/v1/auth/refresh', { refreshToken })
    → AuthController.refresh(@RequestBody body)
      → AuthService.refreshToken(refreshToken)
        → JwtTokenProvider.validateToken(refreshToken)   // 校验是否过期

        → JwtTokenProvider.createAccessToken(userId, tenantId, userType, roles)
        → JwtTokenProvider.createRefreshToken(userId, tenantId)
        ← return Map{ accessToken, refreshToken }
      ← Result.success(tokens)
    → localStorage.setItem('accessToken', newToken)
    → 唤醒排队队列中所有等待的请求
```

**📥 输入**:
```json
{ "refreshToken": "eyJhbGciOiJIUzI1NiJ9..." }
```

**📤 输出**: 同登录（新双Token）

✅ **预期**: RefreshToken 有效 → 签发新双Token → 前端更新 localStorage → 排队请求重试

## 三、订单模块 (campus-order) — 13 个端点

### 3.1 创建订单

```
POST /api/v1/order/orders    Header: X-User-Id: 169..., X-Tenant-Id: 1001
```

**调用链**：
```
OrderController.create(@RequestBody Order order)
  → (从Header读取 userId, tenantId)
  → OrderService.createOrder(order, userId, tenantId)

    → 扣减库存 (Redis DECRBY)
  ← return Order (含订单号、状态、快照)
```

**📥 输入**:
```json
{
  "skuId": 1001, "quantity": 1,
  "addressId": 10, "couponId": null,
  "totalAmount": 88.00, "payAmount": 78.00
}
```

**📤 输出**:
```json
{
  "code": 200, "data": {

    "status": "PENDING_PAY", "totalAmount": 88.00, "payAmount": 78.00,
    "orderSnapshot": "{...JSON快照...}"
```

## 五、优惠券模块 (campus-coupon) — 6 个端点

### 5.1 抢券（高并发核心）

```
POST /api/v1/coupon/grab/{templateId}    Header: X-User-Id: userId
```

**调用链**：
```
CouponController.grab(templateId, userId)
  → CouponGrabService.grab(templateId, userId)
    → Redis EVAL grab_coupon.lua
      → KEYS[1]=coupon:stock:{id}, KEYS[2]=coupon:grabbed:{id}, ARGV[1]=userId
      → SISMEMBER 查重复 → GET 查库存 → DECRBY扣库存 → SADD标记用户
      → 返回: 1=成功 / -1=抢光 / -2=已抢
    → (成功) MQ: coupon.grab.success → 异步持久化到MySQL
  ← switch result: 1→"抢券成功", -1→"已被抢光", -2→"已抢过"
```

**📥 输入**: PathVar `templateId=1`, Header `X-User-Id: 169...`

**📤 输出**: `{ "code": 200, "message": "success", "data": "抢券成功" }`

✅ **预期**: Lua 原子操作 → 扣库存 → 标记用户 → MQ 异步写 MySQL（不阻塞响应）

## 十一、Gateway 安全链路贯穿

### 完整请求生命周期（以"查询我的订单"为例）

```
① 浏览器
   http.get('/api/v1/order/orders?page=1&size=20')
   → Axios 请求拦截器: 自动附加 Authorization: Bearer eyJ...

② Vite Proxy
   localhost:3000/api/v1/order/orders → 转发 → localhost:8080/api/v1/order/orders

③ Gateway (Netty :8080)
   AuthGlobalFilter.filter():
     path = "/api/v1/order/orders"
     isWhiteUrl(path) → false (不在白名单)
     extractToken() → "Bearer eyJ..." → token = "eyJ..."
     jwtTokenProvider.validateToken(token) → true
     jwtTokenProvider.parseToken(token) → UserContext{ userId=169..., tenantId=1001, userType="ADMIN", roles=["ADMIN"] }
     注入 Header:
       X-Gateway-Key: campus-cloud-gateway-2026
       X-User-Id: 1692345678901234567
       X-Tenant-Id: 1001
       X-User-Type: ADMIN
       X-Roles: ADMIN
   → Route to lb://campus-order

④ Order Service (Tomcat :8084)
   GatewayAuthFilter.doFilter():
     path = "/api/v1/order/orders"
     isPublicPath(path) → false
     X-Gateway-Key == "campus-cloud-gateway-2026" → true → 放行

   UserContextFilter.doFilter():
     X-User-Id = "1692345678901234567" → UserContext.set(context)
     chain.doFilter()

   OrderController.myOrders("", 1, 20):
     userId() → request.getHeader("X-User-Id") → 1692345678901234567
     tenantId() → request.getHeader("X-Tenant-Id") → 1001
     → OrderService.myOrders(status, page, size, userId)
       → MyBatis-Plus TenantInterceptor: 自动追加 WHERE tenant_id = 1001
       → OrderMapper.selectPage(page, LambdaQuery.eq(userId))

   UserContextFilter.finally: UserContext.clear()  // 防内存泄漏

⑤ 返回
   Result<List<Order>> → Gateway → Vite Proxy → 浏览器
```
