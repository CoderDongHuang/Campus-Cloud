```java
// 生成Token
public String createAccessToken(Long userId, Long tenantId, List<String> roles) {
    return Jwts.builder()
        .id(UUID.randomUUID().toString())             // jti: 唯一ID，用于黑名单
        .subject(String.valueOf(userId))
        .claim("tid", tenantId)                       // tenantId
        .claim("rol", String.join(",", roles))        // roles
        .issuedAt(new Date())
        .expiration(new Date(System.currentTimeMillis() + 7200000)) // 2h
        .signWith(privateKey)                         // RSA签名
        .compact();
}
```

**关键实现细节**：

1. **登录限流**：
```java
// 同一个IP或手机号，5分钟内失败超过5次，锁定30分钟
public void checkLoginRateLimit(String key) {
    String rateKey = "login:fail:" + key;
    Long count = redisTemplate.opsForValue().increment(rateKey);
    if (count == 1) redisTemplate.expire(rateKey, Duration.ofMinutes(5));
```

### 4.3 智能派单实现

```java
@Service
public class DispatchService {
    
    public List<Long> findBestWorkers(Order order, ProductSpu spu) {
        // 1. 查询可服务区域的在线师傅
        List<WorkerInfo> workers = userServiceClient.getAvailableWorkers(
            spu.getId(), order.getCampusName(), order.getBuildingName());
        
        if (workers.isEmpty()) return Collections.emptyList();
        
        // 2. 计算每个师傅的综合得分
        List<WorkerScore> scores = workers.stream()
            .map(w -> new WorkerScore(w, calculateScore(w, order, spu)))
            .sorted((a, b) -> Double.compare(b.score, a.score))
            .collect(Collectors.toList());
        
        // 3. 取Top3
        return scores.stream()
            .limit(3)
            .map(ws -> ws.worker.getUserId())
            .collect(Collectors.toList());
    }
    
    private double calculateScore(WorkerInfo worker, Order order, ProductSpu spu) {
        double skillMatch = matchSkill(worker.getSkills(), spu.getRequiredSkills()); // 40%
        double loadScore = 1.0 - (worker.getCurrentOrders() / (double) worker.getMaxOrders()); // 25%
        double distanceScore = calculateDistance(worker.getLocation(), order.getAddress()); // 20%
        double ratingScore = worker.getRating() / 5.0; // 15%
        
        return skillMatch * 0.4 + loadScore * 0.25 + distanceScore * 0.2 + ratingScore * 0.15;
    }
}
```

```
   → 发MQ: 通知下单成功、触发派单

3. 退款
   → order-service 调用 payment-service 创建退款单
   → payment-service 调用支付渠道退款
   → 异步回调更新退款状态
```

### 5.3 回调幂等处理

```java
@PostMapping("/callback/{channel}")
public Result<String> callback(@PathVariable String channel, 
                                HttpServletRequest request) {
    // 1. 解析回调参数
    Map<String, String> params = parseParams(request);
    
    // 2. 幂等性检查（基于渠道交易号）
    String channelTradeNo = params.get("trade_no");
    String lockKey = "pay:callback:" + channelTradeNo;
    boolean locked = redissonClient.getLock(lockKey).tryLock(10, TimeUnit.SECONDS);
    if (!locked) return Result.success("处理中");
    
    try {
        // 3. 查询是否已处理
        PayOrder existing = payOrderMapper.selectByChannelTradeNo(channelTradeNo);
        if (existing != null && existing.getStatus() == PayStatus.PAY_SUCCESS) {
            return Result.success("已处理"); // 幂等返回
        }
        
        // 4. 验签
        PaymentChannel channelImpl = channelFactory.get(channel);
        if (!channelImpl.verifyCallback(params)) {
            return Result.fail("验签失败");
        }
        
        // 5. 更新支付单 + 订单状态
        paymentService.handlePaySuccess(channelTradeNo, params);
        
        return Result.success("ok");
    } finally {
        redissonClient.getLock(lockKey).unlock();
    }
}
```

```lua
--[[
  抢券Lua脚本 — 原子操作：检查去重 + 扣库存 + 标记已抢
  
  KEYS[1]: 库存key     coupon:stock:{templateId}
  KEYS[2]: 已抢集合key  coupon:grabbed:{templateId}
  ARGV[1]: userId
  ARGV[2]: count (默认1)
  
  返回值:
    1   — 抢券成功
    -1  — 库存不足
    -2  — 已抢过（去重）
]]

local stock = tonumber(redis.call('get', KEYS[1]) or '0')

-- 检查是否已抢
if redis.call('sismember', KEYS[2], ARGV[1]) == 1 then
    return -2
end

-- 检查库存
if stock <= 0 then
    return -1
end

-- 扣库存 + 标记已抢
redis.call('decrby', KEYS[1], tonumber(ARGV[2]))
redis.call('sadd', KEYS[2], ARGV[1])

return 1
```

```java
@Service
public class CouponGrabService {
    
    private final RedisTemplate<String, Object> redisTemplate;
    private final RabbitTemplate rabbitTemplate;
    private final RedissonClient redissonClient;
    
    // 预加载Lua脚本SHA
    @PostConstruct
    public void init() {
        // 读取 lua/grab_coupon.lua，计算SHA并缓存
        String script = ResourceUtil.readUtf8Str("lua/grab_coupon.lua");
        this.scriptSha = redisTemplate.getConnectionFactory()
            .getConnection().scriptLoad(script.getBytes());
    }
    
    public GrabResult grab(Long templateId, Long userId) {
        String stockKey = "coupon:stock:" + templateId;
        String grabbedKey = "coupon:grabbed:" + templateId;
        
        // 执行Lua脚本
        Long result = redisTemplate.execute(
            new DefaultRedisScript<>(scriptSha, Long.class),
            Arrays.asList(stockKey, grabbedKey),
            userId, 1
        );
        
        if (result == 1) {
            // 抢券成功，异步落库
            MqMessage msg = MqMessage.builder()
                .messageId(IdUtil.fastSimpleUUID())
                .data(Map.of("templateId", templateId, "userId", userId))
                .build();
            rabbitTemplate.convertAndSend(
                "coupon.exchange", "coupon.grab.success", msg);
            return GrabResult.success();
        } else if (result == -1) {
            return GrabResult.fail("优惠券已被抢光");
        } else if (result == -2) {
            return GrabResult.fail("您已经抢过了");
        }
        return GrabResult.fail("系统繁忙");
    }
}
```

```java
@Component
public class CouponGrabConsumer {
    
    @RabbitListener(queues = "coupon.grab.queue")
    public void handleGrabSuccess(MqMessage msg, Channel channel,
                                   @Header(AmqpHeaders.DELIVERY_TAG) long tag) {
        Map<String, Object> data = (Map<String, Object>) msg.getData();
        Long templateId = Long.valueOf(data.get("templateId").toString());
        Long userId = Long.valueOf(data.get("userId").toString());
        
        // 幂等检查
        String idempotentKey = "coupon:grab:record:" + templateId + ":" + userId;
        boolean alreadyProcessed = redisTemplate.opsForValue()
            .setIfAbsent(idempotentKey, "1", Duration.ofHours(24));
        if (!alreadyProcessed) {
            channel.basicAck(tag, false);
            return;
        }
        
        try {
            // 开启本地事务落库
            transactionalTemplate.execute(status -> {
                // 1. 扣减MySQL库存
                couponTemplateMapper.decrStock(templateId);
                // 2. 写入用户优惠券
                UserCoupon userCoupon = buildUserCoupon(templateId, userId);
                userCouponMapper.insert(userCoupon);
                return null;
            });
            channel.basicAck(tag, false);
        } catch (Exception e) {
            // 重试
            channel.basicNack(tag, false, true);
        }
    }
}
```

### 6.3 批量发券

```java
@Service
public class BatchCouponService {
    
    private final ThreadPoolExecutor executor = new ThreadPoolExecutor(
        10, 20, 60, TimeUnit.SECONDS,
        new LinkedBlockingQueue<>(1000),
        new ThreadPoolExecutor.CallerRunsPolicy()
    );
    
    public void batchSend(Long templateId, List<Long> userIds) {
        // 1. 创建发券批次
        CouponBatch batch = new CouponBatch();
        batch.setTemplateId(templateId);
        batch.setTotalCount(userIds.size());
        batch.setSuccessCount(0);
        batch.setStatus("PROCESSING");
        couponBatchMapper.insert(batch);
        
        // 2. 分批多线程发放
        List<List<Long>> partitions = ListUtil.partition(userIds, 100); // 每批100人
        CountDownLatch latch = new CountDownLatch(partitions.size());
        
        for (List<Long> partition : partitions) {
            executor.submit(() -> {
                try {
                    for (Long userId : partition) {
                        sendOneCoupon(templateId, userId);
                    }
                } finally {
                    latch.countDown();
                }
            });
        }
        
        latch.await();
        
        // 3. 更新批次状态
        batch.setStatus("COMPLETED");
        couponBatchMapper.updateById(batch);
    }
    
    private void sendOneCoupon(Long templateId, Long userId) {
        // 分布式锁保证每人每券只发一次
        String lockKey = "coupon:send:" + templateId + ":" + userId;
        RLock lock = redissonClient.getLock(lockKey);
        
        try {
            if (lock.tryLock(3, 10, TimeUnit.SECONDS)) {
                // 检查是否已发
                Long count = userCouponMapper.selectCount(
                    new LambdaQueryWrapper<UserCoupon>()
                        .eq(UserCoupon::getTemplateId, templateId)
                        .eq(UserCoupon::getUserId, userId));
                if (count > 0) return;
                
                // 发券
                UserCoupon coupon = buildUserCoupon(templateId, userId);
                userCouponMapper.insert(coupon);
            }
        } finally {
            if (lock.isHeldByCurrentThread()) lock.unlock();
        }
    }
}
```

```java
                .should(s1 -> s1.match(m -> m.field("name").query(keyword).boost(3.0f)))

                    .findFirst().ifPresent(doc::setHighlightName);
```

```java
@Component
public class ImWebSocketServer {
    
    @PostConstruct
    public void start() throws InterruptedException {
        EventLoopGroup bossGroup = new NioEventLoopGroup(1);
        EventLoopGroup workerGroup = new NioEventLoopGroup();
        
        try {
            ServerBootstrap bootstrap = new ServerBootstrap();
            bootstrap.group(bossGroup, workerGroup)
                .channel(NioServerSocketChannel.class)
                .childHandler(new ChannelInitializer<SocketChannel>() {
                    @Override
                    protected void initChannel(SocketChannel ch) {
                        ChannelPipeline pipeline = ch.pipeline();
                        pipeline.addLast(new HttpServerCodec());
                        pipeline.addLast(new ChunkedWriteHandler());
                        pipeline.addLast(new HttpObjectAggregator(65536));
                        // WebSocket路径: ws://host:9090/ws?token=xxx
                        pipeline.addLast(new WebSocketServerProtocolHandler("/ws"));
                        pipeline.addLast(new ImAuthHandler());      // Token校验
                        pipeline.addLast(new ImMessageHandler());   // 消息分发
                    }
                });
            
            ChannelFuture future = bootstrap.bind(9090).sync();
            log.info("IM WebSocket 服务启动在端口 9090");
            future.channel().closeFuture().sync();
        } finally {
            bossGroup.shutdownGracefully();
            workerGroup.shutdownGracefully();
        }
    }
}

// 消息处理器
@ChannelHandler.Sharable
public class ImMessageHandler extends SimpleChannelInboundHandler<TextWebSocketFrame> {
    
    @Override
    protected void channelRead0(ChannelHandlerContext ctx, TextWebSocketFrame frame) {
        String payload = frame.text();
        ImMessageDTO msg = JSON.parseObject(payload, ImMessageDTO.class);
        
        // 1. 敏感词过滤
        msg.setContent(sensitiveFilter.filter(msg.getContent()));
        
        // 2. 消息路由：发给接收者（在线直接推送，离线存Redis）
        Channel receiverChannel = ChannelManager.get(msg.getReceiverId());
        if (receiverChannel != null && receiverChannel.isActive()) {
            receiverChannel.writeAndFlush(
                new TextWebSocketFrame(JSON.toJSONString(msg)));
        } else {
            // 离线消息存入Redis List
            String offlineKey = "im:offline:" + msg.getReceiverId();
            redisTemplate.opsForList().rightPush(offlineKey, JSON.toJSONString(msg));
            redisTemplate.expire(offlineKey, Duration.ofDays(7));
        }
        
        // 3. 异步持久化到MongoDB
        rabbitTemplate.convertAndSend("im.exchange", "im.message.send", msg);
    }
}
```

| 缓存击穿防护 | Redisson分布式锁控制重建 | 热点数据稳定 |
| 缓存雪崩防护 | TTL加随机偏移 | 避免同时过期 |

## 附录：开发规范与约定

### A. 编码规范

1. **包命名**：`com.sharecampus.{module}.{layer}`
2. **类命名**：
   - Controller: `{Entity}Controller`
   - Service接口: `{Entity}Service`
   - Service实现: `{Entity}ServiceImpl`
   - Mapper: `{Entity}Mapper`
   - DTO: `{Entity}DTO` / `{Action}Request` / `{Action}Response`
   - VO: `{Entity}VO`
3. **方法命名**（Controller层）：查 `get`/`list`/`search`，增 `create`/`add`，改 `update`/`modify`，删 `delete`/`remove`
4. **日志规范**：使用 `@Slf4j`，关键节点打INFO，异常打ERROR（带堆栈）
5. **异常处理**：不吞异常、不 `e.printStackTrace()`、业务异常抛 `BizException`

### E. 关键技术问题FAQ

**Q: 状态机如何新增一个状态？**
A: 
1. 在 `OrderStatus` 枚举新增状态值
2. 新建 `XxxState.java` 实现 `OrderState` 接口
3. 在 `OrderStateFactory` 注册映射
4. 旧代码无需改动

**Q: 如何验证分库分表路由正确？**
A: 查看ShardingSphere日志（开启 `sql-show: true`），确认SQL被路由到正确的库和表。

**Q: 抢券Lua脚本报错怎么办？**
A: Lua脚本错误会返回异常，Java端捕获后返回"系统繁忙"。同时定时对账任务会发现库存差异并修复。

**Q: 分布式事务什么场景用？**
A: 跨服务的强一致性场景，如：下单时需同时扣优惠券+创建支付单+扣库存。使用Seata AT模式。
